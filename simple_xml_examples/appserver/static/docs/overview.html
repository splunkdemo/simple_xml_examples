Getting started

This Simple XML Examples app provides a hands-on way to learn the basic concepts and tools needed to create dashboards using Simple XML. Simple XML is the easiest way to develop dashboards, requiring little or no programming knowledge.

Now that you've installed the app, you have all you need to begin creating your own dashboard.

Besides this Overview this app provides the following navigation tabs:

  • Examples update

    Study the examples to begin creating your own dashboards. Each example gives you a recipe for implementing a dashboard element, beginning with the most basic elements and progressing to more complex, custom elements. You learn basic Simple XML concepts and how to work with the built-in tools. You can review the examples of elements that are most relevant to your project.

    Each example includes an actual runtime visualization followed by a Description that includes:

    • The goal of the example and expected results.
    • The Simple XML parts used in the example and how to implement them.
    • Alternative solutions to consider and recommended next steps.
    • The source code for the example. Cut-and-paste the source code into the Edit Source dialog of a dashboard to get started.

    Elements and techniques in the Examples tab are categorized to make it easy to locate the right element, as well as related elements, for your dashboard. Many of the simple element examples are complemented by variations on the example that show how elements can be extended to provide a richer experience.

  • Dashboards

    This tab lists all dashboards in the Simple XML Examples app. You can create a new dashboard using the built-in Visual Editor and XML Source Editor dashboard GUI. You can experiment with creating your own dashboards, using what you learned in the Examples tab.

  • Search

    This is the common Splunk Enterprise Search dashboard. You can experiment with search strings to see results before including the search in your own dashboard.

How Simple XML fits in the Splunk Enterprise Web Framework

Simple XML is part of the Splunk Enterprise Web Framework. The framework provides several methods for creating rich dashboards as shown in the following taxonomy:

Note: The Simple XML Examples app covers the highlighted parts of the Web Framework taxonomy.

The app creation methods differ in the amount of programming needed to implement a dashboard. As you might expect, more programming gives you more customization options. Simple XML requires minimal programming knowledge, whereas the other methods require programming knowledge, depending on the programming environment and technology complexity.

This Simple XML Examples app shows you how to create dashboards using Splunk Enterprise simple XML and simple XML extensions. Simple XML extensions are ideal if you need a few customizations while retaining the ability to easily edit the dashboard.

The Custom HTML + Splunk Enterprise JavaScript stack and Django bindings are methods available with the Splunk Enterprise Web Framework.

The Module System method employs legacy Splunk Enterprise application framework technologies, including modules and advanced XML.

Choose the method that matches your technology preference and customization requirements. The simple XML solution includes the following capabilities, which gives you powerful facilities for creating rich dashboards:

  • Simple XML

    Simple XML is a simplified markup language for implementing Splunk Enterprise dashboards. Splunk enterprise built-in tools make it easy to create your dashboard:

    • Visual Editor
      A powerful GUI tool for working directly with simple XML.

    • XML Source Editor
      Directly editing the underlying XML to leverage options and configurations not available using the Visual Editor. Directly editing the XML has no effect on your ability to continue to use the Visual Editor or the integrated PDF generation.

    Features

    • Form Inputs
      Add form inputs to your dashboard, such as text inputs, dropdowns, and radio buttons.

    • Advanced Panel Format Options
      Add configuration options to your chart/table/single/event dashboard element that are not supported using the Visual Editor. See the Simple XML view reference for detailed information.

    • HTML Panel
      Add custom HTML as a panel within your dashboard.

    • Dynamic Drilldown
      Construct custom workflows within your app, setting the target of a drilldown event to be another dashboard in your app, and pass attributes from the search results through the URL. Dynamic drilldown is supported by both tables and charts.

    • Maps
      Leverage the Splunk Enterprise integrated maps visualization within your dashboard.

    • Post-process Searching
      Run a single background search and execute multiple post-process searches to drive dashboard panels.

    • PDF Generation
      Generate a PDF of any Simple XML-based dashboard.

    • Preserve Drag-and-Drop
      Continue editing dashboards using the drag-and-drop Dashboard Editor.

    • Auto-Refresh Page
      Configure your dashboard to periodically refresh.

  • Simple XML extensions

    Several Simple XML extension points provide the ability to modify layout, add new visualizations, and customize behaviors for a simple dashboard. The Visual Editor and PDF generation are available with Simple XML extensions.

    Features

    • Stylesheet
      Apply a custom CSS stylesheet to a single dashboard within an app.

    • Layout
      Make simple layout changes, such as rearrange form/dashboard elements and hide panels.

    • Token Setting
      Set custom tokens for use throughout the dashboard page.

    • Custom Viz
      Integrate a custom visualization as a panel.

    • Custom Drilldown
      Customize the drilldown event for tables, charts, and other elements.

    • Table Cell Renderer
      Include custom styles and behaviors within table cells.
