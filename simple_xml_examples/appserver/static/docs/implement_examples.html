Implement the Examples

You can easily incorporate the examples that use simple XML extensions into you own dashboards. Copy the JavaScript and CSS source files from the examples to your app as described in the following procedure.

The procedure details the steps for the Table Cell Highlighting example. The steps are the same for the other simple XML extensions examples.

Note: Perform the steps in the order listed in the procedure to avoid restarting Splunk Web to view the changes.

  1. Copy the JavaScript and CSS source files to your app:

    Source files:

    $SPLUNK_HOME/etc/apps/simple_xml_examples/appserver/static/table_cell_highlighting.js
$SPLUNK_HOME/etc/apps/simple_xml_examples/appserver/static/table_cell_highlighting.css

    Target location:

    $SPLUNK_HOME/etc/apps/[YourApp]/appserver/static/table_cell_highlighting.js
$SPLUNK_HOME/etc/apps/[YourApp]/appserver/static/table_cell_highlighting.css

  2. In your dashboard or form, from the top-level <dashboard> or <form> elements, use the script and stylesheet attributes to access the JavaScript and CSS files. 

    <dashboard script="table_cell_highlighting.js" stylesheet="table_cell_highlighting.css">

  3. Refresh the dashboard.

  4. (Optional) Restart Splunk Web.

    This step is only necessary if you did steps 1 and 2 out of order. From the CLI, issue the following command:

    $SPLUNK_HOME/bin/splunk restart splunkweb
